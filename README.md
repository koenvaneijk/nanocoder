# nanocoder.py

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Python 3.8+](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/downloads/)

A minimal AI coding assistant that runs in your terminal.

## Installation

```bash
curl -O https://raw.githubusercontent.com/koenvaneijk/nanocoder/refs/heads/main/nanocoder.py
chmod +x nanocoder.py
```

## Usage

```bash
python3 nanocoder.py
```
